Support for external devices.

This package provides support for external devices which need extension of the Java Runtime Environment for communication and sometimes also need proprietary interface support software.

Currently the following devices are supported:

  • The GretagMacbeth EyeOne colorimeter. This needs the GretagMacbeth EyeOne SDK library for USB support.
  • The Tektronix Lumacolor II with J1810 colorimeter head. This is connected to the system using a serial communication port (RS232) and requires the installation of the Java Communications API.
  • Other devices connected to the system via the serial communication port (RS232) and also require the installation of the Java Communications API.
